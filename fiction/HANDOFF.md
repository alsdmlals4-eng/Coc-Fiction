## 현재 상태

- 5화 단위 원고 45개 생성

- 화별 제목·POV·분량·SHA 색인 완료
- 활성 원고에서 폐기된 2부 세력·인명 제거 확인
- 다음 단계: 전체 역개요와 대표 화 심화 퇴고

## 작업 절차

현재 5화 원고 → 대응 원본 로그 → `MANUSCRIPT_INDEX.json` 기준 원문 확인 → 장면별 시작 상태·목표·장애·선택·대가·종료 상태 추출 → 구조·연속성 검수 → 승인된 최소 수정 → SHA·Revision Report 갱신 → 회귀 검사 순으로 진행한다.

## 금지

구형 Google Docs 또는 `archive/`를 활성 정본처럼 사용하지 않는다. 별도 압축 초안은 최신 통합 원고보다 오래된 파생본이므로 작업 입력에서 제외한다. 원문 대조가 필요할 때만 `SOURCE_MANIFEST.md`에서 출처 상태를 확인한다.
