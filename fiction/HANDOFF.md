- 225화 구조 역개요 기준선과 부·외전별 진단 완료

- 다음 단계: 제10화·제95화·제180화 파일럿 scene diagnostic과 원본 장면 대조

## 작업 절차

현재 5화 원고 → `REVERSE_OUTLINE_001_225.json`의 해당 화 증거 → 대응 원본 로그 → `MANUSCRIPT_INDEX.json` 원문 SHA 확인 → 장면별 시작 상태·목표·장애·선택·대가·종료 상태 수동 확정 → 구조·연속성 검수 → 비판 재판정 → 승인된 최소 수정 → SHA·Revision Report 갱신 → 회귀 검사 순으로 진행한다.

## 주의

- 자동 역개요는 탐색 기준선이며 원고보다 우선하지 않는다.
- 문단 밀도나 주제어 빈도 플래그를 오류로 자동 판정하지 않는다.
- 원본 장면 대조 전에는 사건 충실도를 `UNVERIFIED`로 유지한다.
- 구형 Google Docs 또는 비활성 과거 자료를 현행 작업 입력으로 사용하지 않는다.
