# 《폭풍의 눈》 Style Guide

## 기본

- 제한적 3인칭 다중 POV
- 같은 장면 안에서 무표식 시점 이동 금지
- 설명보다 행동·감각·선택의 결과를 우선한다.
- 같은 감정을 설명·신체 반응·대사로 세 번 반복하지 않는다.
- 밝은 일상과 만담이 공포·폭력으로 꺾이는 명암 대비를 유지한다.
- 죽음·고문·희생·이별 직후에는 농담으로 감정을 덮지 않는다.

## POV 렌즈

- 주안: 출입구, 사선, 거리, 손, 몸의 반응. 짧고 행동 중심.
- 엘리스: 표정, 침묵, 관계 거리, 감정의 어긋남. 타인의 선택지를 본다.
- 이안: 규칙, 원인, 예외, 문서, 기록. 분석적 불평과 자기비하형 유머.
- 다빈: 일상의 기억, 몸의 이질감, 자기 권리. 보호 대상이 아닌 결정권자.
- 주민: 증상, 손상, 치료 가능성, 동의. 구체적이고 절제된 의료 관찰.
- 엘리엇: 가능한 결과와 실패 확률. 진심과 확신이 통제로 변하는 언어.

## 장면 작성

첫 문단에서 POV·시간·공간·즉시 압력을 붙잡게 한다. 장면은 `압력 → 목표 → 장애 → 선택 → 대가 → 상태 변화 → 다음 압력` 중 필요한 단계를 가지며, 사건·인식·관계·정서 중 최소 하나를 바꾼다.

## 전투

누가 어디에 있고 무엇을 보며 어떤 경로로 움직이는지 보여준다. 승리에는 정보·준비·관계·희생 중 하나 이상의 사전 근거가 있어야 한다. 능력은 이상 징후 → 무의식 발현 → 제한적 성공 → 오사용 피해 → 규칙 발견 → 윤리적 선택 → 팀 조합 → 대가를 감수한 사용의 계단을 따른다.

## 표기

- 엘리스
- 밀리 코사
- 한 장면 POV 표식은 `---` 다음 인물명
- 원고 묶음에서는 `## 제N화 · 소제목`, `**POV:** 인물명` 형식을 사용한다.

## 파일럿 퇴고 규칙

- 엘리스의 보호 반응은 외부 존재나 완전히 분리된 인격으로 확정하지 않는다. 대화 형식이 필요해도 같은 자아 안의 상태·기능·내면 목소리임을 문맥으로 유지한다.
- 전투 후유증은 다음 전환 전에 감각·호흡·부상 중 최소 하나로 남기되, 즉시 진행을 멈추는 별도 설명 장면을 기계적으로 만들지 않는다.
- 외전 종결은 앞 화에서 이미 정한 결심을 다시 요약하기보다, 기한·이동·소지품·메시지 같은 다음 행동으로 닫는다.
- 의료·기관 장면은 환자 동의와 중단 권한을 먼저 세우고 장비 출처·보관자·운용자·증거 책임을 구분한다. 전문 용어는 주민이 지금 판단할 증상과 행동에 연결한다.
- 새 조직명으로 폐기 세력의 협상·장비 공급·회수 기능을 묶어 복원하지 않는다. 같은 장비라도 출처, 보관, 기술 운용, 증거보전, 의료 결정권을 분리해 표기한다.

## 5화 묶음 연속성 규칙

- 앞 화보다 이른 병행 시점으로 돌아가면 첫 문단에 시간 앵커를 둔다.
- POV가 바뀌어도 이미 완료된 승선·이동·교전을 현재 사건처럼 다시 시작하지 않는다.
- 병행 사건은 `같은 시각`만 쓰고 끝내지 말고 서로 다른 위치·목표·확보 정보를 명시한다.
- 묶음 검수는 앞 화와 다음 화를 경계로 포함해 시간·동선·부상·지식·소지품·비밀을 확인한다.
- 문단이나 설명을 늘리기 전에 경계 오류와 중복 이동을 먼저 제거한다.
