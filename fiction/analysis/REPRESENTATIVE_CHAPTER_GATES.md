# 대표 품질 게이트 12화

상태: **ACTIVE REVIEW / PILOT 3 COMPLETE**  
갱신: 2026-07-23

## 목적

전체 225화를 한 번에 문장 퇴고하기 전에 서로 다른 구간·POV·장면 기능을 대표하는 12개 화로 작품 코어와 퇴고 파이프라인의 재현 가능성을 시험한다. 선정은 분량이나 선호도가 아니라 구조적 대표성을 기준으로 한다.

## 공통 게이트

각 화에서 다음을 확인한다.

1. 작품 코어와 장르 약속이 실제 장면에서 느껴지는가
2. POV 렌즈가 행동·정보·감정 선택을 제한하는가
3. 목표·장애·선택·대가·상태 변화가 연결되는가
4. 정보 공개가 공정하며 다음 압력을 만드는가
5. 주제가 설명이 아니라 행동과 결과로 드러나는가
6. 이 품질과 수정 규칙을 다른 화에 반복 적용할 수 있는가
7. 원본 사건·동행·결과와 충돌하지 않는가 — 원본 대조 전에는 `UNVERIFIED`

## 화별 수동 판정

### 제1화 · 출항과 금서 — PASS

- **강점:** 항구의 밝은 감각과 금서의 차가운 반응이 독자 약속을 즉시 만든다. 이안은 장비·문서·브루스의 표정을 통해 이상을 감지한다.
- **위험:** 즉시 목표가 짐 운반과 승선으로 작아 보이지만 개막부 기능상 허용 가능하다.
- **보호:** 여행의 들뜸, 이안의 불평 섞인 관찰, 마지막 불길한 표정.

### 제3화 · 벌꿀주스와 신입 경호원 — PASS

- **강점:** 주안의 시선이 사람보다 출입구·거리·손과 행동을 먼저 읽는다. 엘리스와의 만담 안에 명령과 자발적 판단의 차이가 설치된다.
- **위험:** 복종 징후를 너무 선명하게 보강하면 관계의 불확실성이 사라진다.
- **보호:** 가벼운 첫 만남과 `아직은`의 여운.

### 제10화 · 거짓 무전과 브루고 — PILOT PASS

- **수정:** 엘리스의 보호 반응을 별개 존재가 아닌 같은 자아의 전면 상태로 명료화했다. 주안이 브루고의 뼈를 부순 감각과 정신 공격 뒤 떨림을 남겨 제11화의 육체 전투로 연결했다.
- **보존:** 거짓 무전 판별 → 우회 선택 → 브루고전 → 노란 코트 정신 공격 → 엘리스의 개입 → 선박 붕괴 순서.
- **회귀:** 보호 반응이 지배 능력이나 외부 존재로 바뀌지 않았고, 주안의 선택 기원은 계속 불확실하다.
- **전이 규칙:** 다중 전환 화에서는 새 설명 장면을 추가하기보다 전환 직전 감각·부상 한 줄로 이전 비용을 남긴다.

### 제20화 · 두 주인의 점토판 — CONDITIONAL PASS

- **강점:** 세계관 설명이 `누구의 명령을 따를 것인가`라는 인물 선택으로 연결된다. 제3의 길이 이후 아크의 논리적 기반이 된다.
- **위험:** 점토판·세력·주인 개념이 짧은 범위에 집중된다.
- **다음 검사:** 질문·반론·물리적 행동이 정보 덩어리를 충분히 끊는지 확인한다.

### 제70화 · 응답하지 않는 호출기 — PASS / SHOULD_FIX

- **강점:** 가족 회복 뒤 복종인자 폭로가 관계의 의미를 다시 흔들고, 주안의 이탈이 실제 관계 비용을 만든다.
- **위험:** 선택·복종·설명 책임을 이미 행동으로 보여준 뒤 서술이 여러 번 명시한다.
- **보호:** 주안에게는 임시 침묵이지만 엘리스에게는 버림이라는 비대칭.

### 제95화 · 호수가 보이는 마을의 끝 — PILOT PASS / MUST_FIX RESOLVED

- **수정:** 확장 문단과 구 파생 문단이 같은 결론을 두 번 반복하던 구조를 하나의 장면으로 통합했다. 호수 관찰 → 세 사람의 보고 → 일주일 기한 → 이가레스 답신 → 평범한 실패 → 정류장 선택으로 전진한다.
- **보존:** 조세이칸의 임시 한국 체류, 아킴의 신앙과 방법 수정, 주안의 미완성 상태, 기억을 지우지 않는 호수, 직접 고른 다음 장소.
- **회귀:** 호출기는 음성·문자를 보내지 않으며, 녹음 파일은 별도 저장장치에 남는다.
- **전이 규칙:** 부 결말은 앞 화의 결심을 다시 요약하지 않고 기한·이동·소지품 같은 다음 행동으로 닫는다.

### 제130화 · 기다리지 않는 호출 — PASS / SHOULD_FIX

- **강점:** 엘리스가 조직의 힘으로 상대를 끌어오지 않고 한 번의 신호만 남기는 행동이 레드퀸 아크를 정확히 닫는다.
- **위험:** 행동 자체가 충분히 강하므로 이후 설명이 같은 의미를 반복할 수 있다.
- **보호:** 기다리되 강요하지 않는 연결 방식.

### 제165화 · 미래를 기록하는 사람 — PASS

- **강점:** 엘리엇의 진심과 성공이 통제 논리로 변하는 과정이 확률·기록·약속의 언어 안에서 드러난다. 비극이 단순 악역화되지 않는다.
- **위험:** 주제 문장이 직접적이지만 인물의 누적 행동으로 획득한 결론이므로 일괄 삭제하지 않는다.
- **보호:** 사람을 살리겠다는 약속과 선택을 대신하겠다는 선언 사이의 균열.

### 제166화 · 눈 속의 오두막 — PASS / CONTINUITY CHECK

- **강점:** 눈·오두막·옛 친구의 대면이 2부의 제도적 원죄를 독립된 호러 장면으로 설치한다.
- **위험:** 윌리엄 단독 POV와 과거 시점이 제167화의 주민 일상으로 즉시 전환된다.
- **다음 검사:** 오두막에서 남긴 변수가 제167화~제170화 안에서 독자가 인식할 수 있는 압력으로 회수되는지 확인한다.

### 제180화 · 몸 안의 멈춘 시간 — PILOT PASS / CANON REPAIR

- **수정:** 환자 중단 버튼과 의료 중단 기준을 먼저 세우고, 장비 출처·보관·공동 운용·증거봉인을 분리했다. 사망 기록은 실제 과거와 선택되지 않은 결과를 구분하지 못한다는 한계를 명시했다.
- **보존:** 다빈에게 원문 결과를 돌려주는 주민, 두 상반된 사망 기록을 함께 남기는 태도, 장비 폭주 뒤 현재 맥박부터 확인하는 의료 렌즈, 열네 살 주민의 단서.
- **회귀:** 폐기된 장기 조직의 장비 공급·연구 소유 기능을 제거하고 현행 2부 바이블의 권한 분산을 적용했다.
- **전이 규칙:** 의료 장면은 `동의·중단 → 검사 → 이상 → 기관 충돌 → 현재 몸 확인` 순서로 독자가 판단 기준을 잃지 않게 한다.

### 제200화 · 창을 지키는 사람 — PASS

- **강점:** 다빈의 주체성이 설명이 아니라 지켜야 할 위치와 공격 경로를 스스로 선택하는 행동으로 드러난다. 승리보다 목적을 지키는 전투다.
- **위험:** 다중 팀의 위치가 흐려지면 다빈의 선택이 다른 세력의 구조에 흡수될 수 있다.
- **보호:** 다빈이 보호 대상이 아니라 결정을 내리는 전투 주체라는 점.

### 제225화 · 네가 없는 마을 — PASS / SHOULD_FIX

- **강점:** 이안이 엘리엇 개인의 죄와 기관의 책임을 기록으로 분리하고, 미래를 끝까지 확인하려는 습관을 멈추며 결말 방향을 완성한다.
- **위험:** 청문회, 여러 인물의 이후 상태, 기관 책임, 작품 명제가 한 화에 모여 요약 보고서처럼 느껴질 수 있다.
- **보호:** 변호문도 사형선고도 아닌 기록, 쓰이지 않은 다음 장.

## 파일럿 3화 결과

세 화 모두 사건·결말 방향을 바꾸지 않는 최소 수정으로 통과했다. 제95화의 중복 종결과 제180화에서 발견된 2부 Canon 전파 오류는 `MUST_FIX`로 처리했다. 제10화의 다중 전환은 삭제·분할보다 보호 반응의 정체성과 후유증 연결을 명료화하는 쪽이 적절했다.

다음 작업은 `006-010`, `091-095`, `176-180`의 나머지 화를 같은 기준으로 검토하되, 파일럿 문장 구조를 기계적으로 복제하지 않는다. 원본 본체가 제공되기 전 사건 충실도는 `UNVERIFIED`다.
