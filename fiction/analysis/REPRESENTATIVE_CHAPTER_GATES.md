## 공통 게이트

## 완료된 대표 수정

### 제10화 — BUNDLE PASS

- 같은 자아의 보호 반응, 브루고전의 육체 비용, 정신 공격 뒤 떨림을 보존했다.
- 제5화의 황색 승선과 제6화 식당을 병행 시간선으로 명료화했다.
- 제7·8화 직접 연속, 제9화 병행 납치, 제10·11화 직접 연속을 확인했다.

### 제95화 — PILOT PASS

- 중복 외전 종결을 하나의 장면 흐름으로 통합했다.
- 호출기와 별도 음성 저장장치의 기능을 분리했다.

### 제180화 — PILOT PASS / CANON REPAIR

- 환자 동의와 중단 기준을 장비 설명보다 먼저 두었다.
- 장비 출처·보관·운용·증거·의료 권한을 분산했다.

## 보호 규칙

- 대표 화 판정을 전체 원고의 자동 수정 명령으로 사용하지 않는다.
- 파일럿·묶음 규칙은 동일 결함이 확인된 화에만 적용한다.
- 원본 Drive 폴더가 접근 가능해지면 제6~10화부터 직접 대조한다.
