## 경계 연속성

| 경계 | 확정 |
|---|---|
| 제90→91화 | 제90화는 한국 도착. 제91화 첫 문단이 ‘한국 도착 첫날 밤’과 ‘비행 전날 회상’을 동시에 명시하므로 시간 역행이 무표지로 보이지 않는다. |
| 제91→92화 | 회상이 끝나 현재 한국 생활로 복귀한다. |
| 제92→93화 | 같은 2018년의 다른 인물 에필로그로 전환하며 POV를 엘리스로 표기한다. |
| 제93→94화 | 엘레인이 번호를 전달하고 엘리스가 누른 전화가 직접 이어진다. |
| 제94→95화 | 전화가 끝난 호텔 외부로 공간을 이동하고 POV를 다빈으로 명시한다. |
| 제95→96화 | 제95화는 외전1과 현재 시점의 다음 이야기 예고를 닫는다. 제96화 첫 문장이 ‘시간은 다시 여덟 해 전’이라고 선언해 외전2 과거편을 연다. |
