## 작업개요 / 전체 설계
## 작품 전체를 감독하는 탭입니다. 설정·인물·에피소드·씬을 수정할 때마다 전체 로그라인, 전체 테마, 전체 플롯을 먼저 재검토하고 변경점이 있으면 사건 타임라인·복선·에피소드 목차까지 함께 갱신합니다.

## 1. 로그라인
- 항목
내용
- 전체 로그라인
태평양 한가운데의 기묘한 섬에 표류한 전직 국가대표 경호원 주안, 카터 가문의 정신계 능력을 깨우는 엘리스, 금기의 고서와 연결된 미스캐토닉 학생 이안은 섬·황색 교단·델타그린의 전쟁 속에서 각자의 힘과 과거를 얻는 대신, 자신들의 선택과 감정마저 누군가의 계획이었는지 의심하게 된다.
  - Ep.8
- 장르·형태
현대 크툴루 신화 기반 생존 미스터리에서 세력전과 능력 활극으로 확장되는 13부 장편. 초반은 일상과 공포, 중반은 탐사와 세력 충돌, 후반은 각성과 전쟁에 무게를 둔다.
- 주인공 목표
초반에는 선상 재난에서 살아남고 서로 재회한다. 중반에는 납치된 사람과 동료를 구하며 섬의 규칙과 세 세력의 목적을 해독한다. 후반에는 성맥과 의식을 둘러싼 계획을 저지하고, 타인이 설계한 힘을 자신의 선택으로 바꾼다.
- 핵심 장애
폐쇄된 섬과 어인의 생존 논리, 황색 교단의 인질·탈취 작전, 윌리엄 카터의 각성 설계, 그리고 성맥·정신계 능력·마법이 남기는 육체와 정신의 비용.
- 독자 약속·결말 방향
공포의 법칙을 알아갈수록 세 주인공은 더 능동적이고 강해지지만, 승리는 관계와 인간성의 대가를 남긴다. 결말은 델타그린의 승리, 섬의 패배, 황색의 부분 승리와 세 주인공의 분리된 성장으로 끝난다.

## 2. 전체 테마
- 항목
내용
- 메인 테마 질문
사랑·보호·대의가 타인의 선택을 빼앗는 순간에도 선의라고 할 수 있는가?
  - Ep.8
- 핵심 명제
괴물성은 외형이나 힘이 아니라, 상대를 목적과 자원으로 취급하며 선택권을 빼앗는 순간 시작된다.
- 주안의 테마
만들어진 복종과 자발적인 보호를 구분하고, 강한 육체가 인간성을 지우지 못하도록 스스로를 통제한다.
- 엘리스의 테마
보호받는 가문의 후계자와 계획된 각성체라는 자리에서 벗어나, 자신의 힘과 관계를 직접 선택하는 레드퀸으로 성장한다.
- 이안의 테마
알고 싶다는 욕망을 감당할 책임으로 바꾸고, 금기의 지식을 실제로 사람을 구하는 수단으로 완성한다.
- 세력 테마·결론
섬·황색·델타그린은 모두 자신을 수호자라고 여기지만 다른 존재를 희생한다. 완전한 선은 없으며, 끝까지 타인의 선택을 남겨두는 태도가 인간성을 증명한다.

  - 생존 판단과 세력전 속에서도 팀 농담을 유지하되 실패와 부상을 누적
  - 능력 활극의 통쾌함을 주되 상처·희생·책임을 함께 회수
  - 처음부터 끝까지 같은 강도의 비장함이나 개그를 유지하지 않는다.
- 유머
인물의 관계와 방어기제로 사용한다. 위험 신호가 확인되면 농담은 즉시 멈춘다.
  - 브루스의 과잉 설명, 주안·엘리스의 신경전, 이안의 피곤한 독백
  - 아킴·페스타의 기괴한 유머와 주연들의 생존형 농담
  - 최종전 직전 자기희생 논쟁에서 각자의 농담이 결의로 바뀜
  - 피해자의 죽음과 고통 자체를 웃음거리로 만들지 않는다.
- 잔혹함
행동과 결과를 구체적으로 보여주되 고어를 목적화하지 않는다.
  - 아킴의 외부 임무로 잔혹성을 먼저 제시하고, EP2 선상 공격에서 그 위협이 주연과 충돌한다.
  - 비살상 시도→불가능 판단→필요한 살상과 후유증
  - 괴물전은 규모와 육체 파괴를 확대하되 주연의 인간성 선택을 남김
  - 잔혹한 결과 뒤 아무 감정·관계 변화가 없는 장면을 피한다.
- 내면 독백
각 인물의 판단 오류와 욕망을 드러낸다.
  - 이안의 호기심과 자기합리화, 주안의 경호 계산, 엘리스의 품위와 경계
  - 비밀·의심·죄책감이 팀 갈등으로 연결
  - 각자의 능력과 희생 논리를 스스로 정당화하는 독백
  - 작가 설명을 주인공 독백으로 위장하지 않는다.
- 정보 공개
고유명사는 제한하지 않는다. 각 씬에서 사용한 인물·조직·장소·유물은 검수 메모 바로 아래의 고유명사 문단에 모아 짧게 정의한다.
  - 장면 원고는 전체 폭으로 읽고, 장면이 끝난 뒤 고유명사와 현재 기능을 한 번에 확인하도록 구성한다.
  - 중후반 세력·인물·장소도 같은 형식으로 누적해 독자의 기억을 지원한다.
  - 노덴스·성맥·최종전 장치는 첫 사용 씬 아래에서 설명하고 이후에는 반복 설명을 줄인다.
  - 고유명사 자체를 줄이기보다 설명 누락·기능 없는 이름 나열·본문 옆 병렬 설명을 피한다.
- 전투/공포 리듬
관찰→계산→선택→짧은 행동→물리적 결과→다음 위험 순서.
  - 첫 살상은 짧고 갑작스럽게, 인물의 반응은 더 짧게
  - 거리·장비·부상·탈출로를 구체적으로 기록
  - 능력은 초반 복선과 대가를 근거로 압도적으로 사용
  - 의성어 연속 사용과 장황한 기술 설명을 피한다.
- 삽화
도입은 상징과 관계, 후반은 능력과 전장 규모를 잡는다.
  - 출항·고서와 펜던트·벌꿀주스와 도박장·빗속의 아킴 임무
  - 섬의 기록·제물 행렬·드림랜드·지도 탈취
  - 푸른 불꽃의 눈·어인의 창 원거리 공명·진명이 공개되는 창 노획·3트랙 최종전
  - 괴물 전신만 반복하지 않고 인물 감정과 상징물을 우선한다.
- 파일/일차 혼합
원본 로그 파일의 경계보다 장면의 인과와 감정 흐름을 우선한다.
  - EP1부터 1일차 중심 사건에 2일차 선상재난 전조를 일부 선행 배치한다.
  - 중반부는 분산된 전투·탐사·세력 정보를 기능별로 재조합한다.
  - 후반부는 각성·계승·최종전 사건을 클라이맥스 호흡에 맞춰 교차한다.
  - 각 씬에 원본 로그 출처 범위를 표시해 추적 가능성을 유지한다.

## 9. 삼세력 비밀 인과와 결말 설계

- 0. 사건 이전
황색 대주교가 모종의 거래와 반쯤 휴가를 위해 델타그린에 자발적으로 잡혀준다.
  - 황색 대주교 / 델타그린
  - 대주교는 식인과 괴이한 의식을 당연시하는 교단 문화에서 잠시 벗어나 거래를 진행하고, 델타그린은 황색 최고 전력을 억류한다.
  - 황색 교단의 지도자가 델타그린에 붙잡혀 있음
  - 포획 자체는 대주교의 선택이지만 거래 전모는 끝까지 밝혀지지 않는다.
  - 후반 일부 공개
  - 거래 내용은 맥거핀
- 1. 판 배치
카터가 주안과 엘리스를 섬 사건에 진입시키도록 상황을 유도한다.
  - 윌리엄 카터
  - 성맥·어인의 창·정신계 개화를 한 사건에 겹쳐 블랙킹과 엘리스의 각성을 유도한다.
  - 가족 여행, 우연한 경호 배정, 선상재난
  - 주안의 배정과 사건 진입 모두 카터의 계산에 포함
  - 중후반
  - 카터의 오산을 남긴다.
- 2. 인질 작전
황색 교단이 엘리스 또는 세실리아를 생포하려 한다.
  - 황색 교단
  - 대주교와의 인질 교환, 카터 혈통 확보, 협상 우위 확보
  - 카터 가문에 대한 복수와 단순 납치
  - 실제 핵심 목적 중 하나는 대주교 회수
  - EP2~10 단서 / 후반 공개
  - 생포 명령을 반복해 복선화
- 3. 삼세력 충돌
섬은 성맥과 어인의 창을 지키고, 황색은 대주교와 성맥을 노리며, 델타그린은 각성 결과를 통제하려 한다.
  - 섬 / 황색 / 델타그린
  - 생존 / 구출·탈취 / 통제·병기화
  - 서로 다른 이유로 싸우는 전쟁
  - 주연 세 사람이 세 세력의 목표가 교차하는 중심이 됨
  - 중반
  - 선악보다 이해관계 충돌로 표현
- 4. 카터의 완성안
주안이 성맥의 육체와 어인의 창 마법을 모두 얻어 블랙킹이 된다.
  - 윌리엄 카터
  - 엘리스의 정신계 능력과 복종인자로 묶인 최강 전력 완성
  - 주안·엘리스의 각성이 위기 속에서 자연스럽게 발생하는 듯 보임
  - 카터의 계획은 실제로 완성되지 않음
  - EP11~13
  - 블랙킹은 계획명
- 5. 분할 각성
주안은 성맥의 육체만 얻고, 대전사장 쿠바라의 어인의 창은 엘리스의 능력적 각성과 강해지고 싶다는 집념에 응답해 옛 신의 마법 권능을 넘긴다.
  - 주안 / 엘리스 / 쿠바라의 유산
  - 각자 살아남고 동료를 지키려는 선택과 엘리스의 집념이 카터의 단일 병기 계획을 깨뜨린다.
  - 두 사람이 함께 최종전을 돌파
  - 통제 가능한 블랙킹 대신 성맥의 육체와 어인의 창 권능이 두 인물에게 갈라진다.
  - EP13
  - 구체적인 선택 조건은 없다. 창의 반응은 엘리스의 각성과 집념에 대한 신화적 공명으로 표현한다.
- 6. 대주교 석방
최종전 전후 인질 교환 또는 기존 거래 이행으로 대주교가 풀려난다.
  - 황색 대주교 / 델타그린
  - 교단은 지도자를 회수하고, 대주교는 휴가가 끝난 듯 담담하게 다음 행보를 택한다.
  - 전쟁 종료 후 황색 지도자 귀환
  - 진짜 결말 갈등이 시작되지만 거래 조건은 설명되지 않는다.
  - 결말 직전
  - 구체 거래는 맥거핀
- 7. 진실 폭로
대주교가 주안에게 블랙킹 계획과 복종인자, 이끌림의 인위적 요소를 알려준다.
  - 황색 대주교
  - 사실을 담담하게 알려주고 주안의 선택을 지켜본다. 처음부터 두 사람을 갈라놓기 위해 모든 일을 설계한 것은 아니다.
  - 진실을 알려주는 기묘한 조력자처럼 보임
  - 사실 공개가 결과적으로 주안과 엘리스를 갈라놓는다.
  - 최종전 직후
  - 대주교를 전지적 흑막으로 만들지 않는다.
- 8. 이별·잠입
주안은 대주교의 폭로 직후 복종인자와 자신의 감정을 구분하지 못하고, 엘리스가 진실을 알고 있었을 가능성까지 의심한 채 떠난다.
  - 주안 / 황색 대주교
  - 성배 후유증 치료, 실험 기록 확인, 감정의 진위 검증, 황색 내부 감시
  - 엘리스를 버리고 황색으로 간 것처럼 보임
  - 주안은 황색 교단원이 아니라 대주교의 손님이자 반쯤 공공연한 내부 감시자다.
  - 에필로그
  - 설명 부재는 계산된 오해가 아니라 감정적 혼란과 불신의 결과
- 9. 레드퀸 육성
윌리엄은 분할 각성을 또 다른 성공으로 평가하고 엘리스를 레드퀸으로 육성한다.
  - 엘리스 / 윌리엄 / 라르고
  - 엘리스는 주안을 다시 찾고 오해를 풀기 위해 델타그린의 육체 훈련과 라르고의 이능 훈련을 받는다.
  - 아버지의 계획을 이용해 강해지는 추적자
  - 엘리스는 본편에서 주안에게 한 번도 정신적 명령을 내리지 않았으며, 재회 후에도 그의 자유 선택을 요구한다.
  - 후일담 / 차기부
  - 윌리엄은 추적을 훈련 동기·주안 회수·황색 정보 접근 수단으로 이용
- 10. 히든카드
라르고 휴빙은 다른 임무를 수행하면서 윌리엄의 직접 지시로 섬 상황과 주연들의 각성을 관찰한다.
  - 윌리엄 카터 / 라르고 휴빙
  - 엘리스 생존, 주안의 회복 가능성, 성맥·어인의 창의 황색 완전 탈취 방지라는 세 조건이 유지되는 동안 불개입
  - 카터가 마지막까지 여유를 잃지 않는 이유
  - 세 조건 중 하나가 무너지면 윌리엄이 라르고 투입을 검토한다.
  - 후반 암시
  - 불개입 임계선 확정
- 11. 주안의 붕괴 위험
성맥의 육체를 얻은 주안은 감각과 감정이 점차 무뎌지고, 실신하거나 감정이 격해질 때 괴물화한다.
  - 주안 / 성맥 / 복종인자
  - 강한 육체의 대가를 인간성의 약화로 남기고, 정신 훈련으로 복종인자를 거스르는 과정을 차기 성장축으로 만든다.
  - 전투에서 회복력이 강해진 각성자
  - 광기에 완전히 잠식되면 괴물 상태에서 돌아오지 못하는 비가역 위험
  - EP13 이후
  - 평상시에는 정상. 실신·감정 폭발·광기가 단계적으로 위험을 높인다.
- 12. 미스캐토닉 완성
이안은 노덴스의 고서의 정식 소유자이자 마법 권총 이든의 가지의 소유자가 된다.
  - 이안 / 미스캐토닉 대학

  - 자연재해형 외적 클라이맥스·정치 협상·부녀 윤리 대면
  - 쇼거스가 최종보스로 기능하고, 전투 승리가 가론·윌리엄 협상과 엘리스의 책임 선언을 발생시키는가

  - 원본 결말의 가족 재회가 충분히 성립한 뒤 각색 반전이 더 큰 낙차를 만드는가
  - EP13 S10~S12

  - 쇼거스 세 분열체와 붉은 핵 / 가론과 윌리엄 사이의 엘리스 / 가족에게 달려가는 엘리스와 주안 / 서로 다른 방향으로 떠나는 두 사람

폭풍의 눈 1부의 사건 결과와 분기점을 먼저 복원한다. 네가 없는 마을 2부는 이 다빈·안 주민·엘리스 3인을 주인공으로 고정하고, 세 사람의 선택·관계·대가를 중심으로 재구성한다. 원본 날짜·파일 경계나 기존 EP 수는 고정하지 않으며, 사건 밀도와 전환점에 따라 EP를 분할·통합·추가·삭제할 수 있다. 폐기된 늑대인간 개인 서사는 사용하지 않는다.
- 확인된 구조 오류
5씬 고정 구조가 사건을 합치거나 삭제했고, 추가한 괴이·복선이 원본의 해안 협상·밀리와 이안의 감정·카터 부녀 대면·쇼거스 최종전을 밀어냈다.
- 복원 순서
1) EP3 해안가 사건 2) EP12 제3세력·쇼거스 협상·창 노획 3) EP13 쇼거스 최종전·가론/윌리엄 협상·부녀 대면·가족 재회 4) EP1~EP11 전수 대조
- 완료 기준
모든 중요 사건에 원본 회차·선택·결과·후속 영향이 연결되고, 삭제·통합한 사건은 이유가 원본사건감사 탭에 기록된다.

2-2. 주연 캐릭터 갈등/화해 패턴
- 상황
주안

  - 화해/정리 방향

- 이안이 몰래 금기를 조사하고 들킴
조용히 경고한다. 단서가 유용했다면 이해하지만, 다음부터는 혼자 움직이지 말라는 식으로 선을 긋는다.
  - 겉으로는 이해하는 척하지만 속으로는 의심이 생긴다. 이안의 호기심이 언젠가 모두를 위험하게 할 수 있다고 본다.
  - 사과하지만, 그래도 단서를 얻었다고 말한다. 죄책감과 합리화가 동시에 있다.
  - 이안의 지식욕과 팀의 안전 사이 충돌
  - 단서가 실제로 도움이 되는 장면을 배치하되, 이후 이안이 정보를 공유하는 습관을 배우게 한다.
  - 사용자 답변 반영
- 엘리스가 카터 가문의 진실을 숨긴 사실이 드러남
겉으로는 괜찮다고 하지만 속으로는 약간 실망한다. 바로 추궁하기보다 이후 판단에 조심스러움이 생긴다.
  - 사과하고 반성한다. 변명보다 자신의 두려움 때문에 숨겼다는 점을 인정하는 방향.
  - 감정적으로 몰아붙이기보다 정보를 우선 정리한다. 위험의 구조를 파악하려 한다.
  - 신뢰와 정보 은폐의 충돌
  - 엘리스의 사과 이후, 주안은 감정 대신 행동으로 다시 곁에 서고 이안은 정보 정리로 협력한다.
  - 사용자 답변 반영
- 주안이 자기 몸 상태를 축소해서 말한 사실이 들킴
사과하지만, 어쩔 수 없는 상황이라면 계속 싸우려 한다. 자신의 상태보다 당장의 보호를 우선한다.
  - 더 걱정하며 울컥한다. 감정이 먼저 나오고, 이후 주안을 멈추게 할 방법을 찾으려 한다.
  - 이성적으로 분석하고 상황에 따라 행동한다. 주안을 쉬게 할지, 전투에 투입할지 판단하려 한다.
  - 주안의 자기희생과 동료들의 보호 욕구 충돌
  - 주안의 희생을 미화하지 말고 후유증과 엘리스의 감정 반응을 반드시 남긴다.
  - 사용자 답변 반영
- 한 명을 미끼로 써야 하는 작전
엘리스가 미끼가 되는 것을 막고 자신이 미끼가 되겠다고 한다.
  - 자신이 나서겠다고 한다. 책임지는 위치이며 자신도 약하지 않다고 주장한다.
  - 다른 변수를 찾는다. 확률상 누군가가 맞더라도 아무도 미끼로 쓰고 싶어 하지 않는다.
  - 보호, 책임, 합리성의 삼각 충돌
  - 셋 모두 선의로 반대하므로 감정싸움보다 각자의 책임 논리로 대립시킨다.
  - 사용자 답변 반영
- 아킴이 필요한 정보를 주지만 대가를 요구함
거래를 받아들인다. 단, 이후 필요하면 뒤통수를 칠 생각까지 포함한다.
  - 협상한다. 약속의 범위와 문구를 제한하고, 가능한 한 대가를 줄이려 한다.
  - 계약의 허점을 찾는다. 마법적/언어적 함정 여부를 우선 의심한다.
  - 생존을 위한 비윤리적 거래와 신뢰 문제
  - 주안은 행동파, 엘리스는 협상가, 이안은 조항 분석가로 역할 분담 가능.
  - 사용자 답변 반영
- 조종당하는 무고한 주민들이 공격해옴
비살상을 노리지만 필요하거나 급하면 죽인다.
  - 조종을 푸는 방법을 찾는다. 방법이 없다면 처리하지만, 책임감과 죄책감이 남는다.
  - 조종을 푸는 방법을 찾는다. 방법이 없다면 처리한다.

  - 죽일 수밖에 없는 상황을 만들더라도 원고에서는 가볍게 넘기지 말고 여파를 남긴다.
  - 사용자 답변 반영

자신은 언제나 살아남을 자신이 있다고 농담스럽게 말하며 남겠다고 한다.
  - 자신이 책임지는 위치이며, 자신도 강해졌다고 주장한다.
  - 자신의 마법적 능력과 지식을 근거로 자신이 남아야 한다고 주장한다.
  - 세 사람 모두 자기희생의 논리가 있음
  - 각자의 주장을 모두 설득력 있게 만든 뒤, 최종 선택은 작전 논리와 감정 논리가 동시에 맞아야 한다.
  - 사용자 답변 반영

## 9. 캐릭터별 말투·유머·잔혹성 표현 기준
- 캐릭터

  - 위기 직전
  - 결정/전투 시 말투
  - 잔혹한 결과 이후
  - 금지 표현
  - 확정 상태
- 주안
자기 처지와 업무를 느긋하게 농담한다. 해고 가능성도 대수롭지 않게 받아들이며 상대의 긴장을 풀어준다.
  - 표정은 크게 변하지 않지만 자세와 중심이 선수처럼 낮아진다. 출구보다 상대의 발과 무게중심을 먼저 본다.
  - 존댓말은 유지하되 지시는 편하고 짧다. “잠깐 뒤에 계세요.” “이건 제가 할게요.”
  - 결과를 과장해 후회하지는 않지만, 자신이 힘 조절에 실패했는지 먼저 확인한다. 과거 결승전의 기억이 짧게 겹친다.
  - 군인식 FM 경호원, 규정 강박, 직업적 충성심 과장, 항상 딱딱한 명령조
  - 사용자 정정
- 엘리스 카터
품위를 유지하면서도 호감을 비교적 솔직하게 드러낸다. 주안에게는 외모·배정·계약을 소재로 장난을 건다.
  - 미소와 말수가 줄고, 시선을 피하지 않은 채 상대 의도를 확인한다.
  - 협상과 명령을 먼저 사용한다. 주안을 지킬 때는 보호 대상이라는 위치를 거부하고 직접 나선다.
  - 무고한 피해가 생기면 책임을 자기 쪽으로 끌어오며 해결책을 찾는다.
  - 주안에 대한 호감을 복종인자의 결과처럼 묘사, 수동적 비명 역할, 귀족식 오만함만 강조
  - 사용자 정정
- 이안 패닝턴
분석과 자기합리화로 불안을 감춘다. 브루스의 기행과 위험한 지식을 현실적인 불평으로 받는다.
  - 말이 길어지다가 핵심 패턴을 발견하면 갑자기 조용해진다.
  - 가능성과 대가를 설명한 뒤 결론을 낸다. 필요하면 금기를 사용하지만 책임을 언급한다.
  - 겉으로는 원인을 분석하고, 속으로는 자기 선택이 만든 피해를 오래 기억한다.
  - 겁쟁이 개그만 반복, 천재라서 모든 답을 즉시 아는 묘사
  - 1차 확정
- 아킴

## 10. 주안·카터 가문 비밀 설정

- 태권도 국가대표
주안은 국가대표 결승전에서 상대 선수에게 중상을 입혀 불명예 은퇴했다.
  - 힘 조절 실패가 아니라 개조된 신체 능력이 순간적으로 튀어나온 사건. 일반 선수의 타격 범위를 명백히 넘어섰다.
  - 자세·거리 감각·하체 중심 이동만 자연스럽게 보여주고 과거 설명은 최소화한다.
  - 상대 선수의 부상 기록과 경기 영상이 다시 언급된다.
  - 주안이 괴물과 싸울 때 같은 출력이 의식적으로 재현된다.
  - 결승전 사고를 단순 폭력 성향으로 오해하게 만들지 않는다.
  - 확정
- 첫 경호 업무
은퇴 뒤 방황하던 중 경호 일을 하던 선배에게 연락받아 처음 현장에 투입됐다.
  - 윌리엄 카터가 주안의 생존을 확인한 뒤, 의심을 피하려 선배를 중간에 세워 영입했다.
  - 규정과 현장 관행을 완전히 알지 못해 느긋하게 행동하며, 잘리면 그만이라고 생각한다.
  - 선배가 실제로 카터의 지시를 받았다는 흔적을 남긴다.
  - 주안은 자기 인생의 재출발마저 설계된 것이었음을 알게 된다.
  - 초보라서 무능한 것이 아니라 경호 규정만 낯설고 신체 판단은 뛰어나다.
  - 확정
- 엘리스의 선택
엘리스는 경호원 이력서 사진 중 주안이 잘생겨 마음에 든다며 가까운 배정을 부탁했다.
  - 카터는 딸의 부탁을 허용하는 척하며 이미 주안을 엘리스 근처에 둘 계획이었다.
  - 엘리스가 주안을 보고 이미 사진으로 골랐다고 장난스럽게 밝힐 수 있다.
  - 배정 과정이 우연이 아니었다는 사실이 드러난다.
  - 엘리스는 아버지가 자신의 호감까지 이용했다는 사실에 분노한다.
  - 엘리스를 철없는 외모지상주의자로만 만들지 말고, 첫 호감의 가벼움과 이후 신뢰를 분리한다.
  - 확정
- 성배 의식 / 검은물
섬에서 카터 가문이 준비한 의식과 검은물이 존재한다.
  - 주안은 성배 의식의 용기 또는 매개체로 쓰기 위해 인위적으로 개조된 실험체였다.
  - 검은물, 잔, 복종 같은 단어는 직접 설명하지 않고 감각적 반응과 악몽으로만 암시한다.
  - 주안의 신체 이상과 카터 가문 연구 기록이 연결된다.
  - 주안이 의식의 도구가 될지 스스로 목적을 선택할지 결정한다.
  - 설정 설명이 주안의 인간성을 압도하지 않게 한다.
  - 확정
- 복종인자
주안은 엘리스에게 이유를 설명하기 어려운 이끌림과 보호 충동을 느낀다.
  - 카터 혈족을 우선하고 명령에 반응하도록 설계된 신체·정신적 편향이 남아 있다.
  - 명령을 들으면 잠깐 몸이 먼저 반응하거나, 엘리스가 위험할 때 판단보다 움직임이 앞서는 정도로 표현한다.
  - 엘리스의 말에 유난히 반응하는 이유가 실험 기록과 연결된다.
  - 주안이 복종 충동을 거스르고도 엘리스를 자기 의지로 선택하는 장면이 필요하다.
  - 복종인자가 사랑 전체의 원인처럼 보이면 관계가 훼손된다. 강제 명령이 아니라 우선순위 편향으로 제한한다.
  - 확정
- 엘리스의 감정
엘리스는 주안에게 순수한 호감과 이끌림을 느낀다.
  - 복종인자와 실험체 설정을 전혀 모르며, 자신의 감정은 조작되지 않았다.
  - 외모에서 시작한 관심이 주안의 편안함과 보호 방식에 대한 신뢰로 변한다.
  - 주안의 반응이 인위적일 수 있다는 사실을 알게 되어 관계를 의심한다.
  - 주안에게 복종이 아닌 명시적인 선택을 요구하고, 자신도 아버지의 설계에서 벗어난다.
  - 엘리스가 죄책감 때문에 일방적으로 관계를 끊는 전개만 반복하지 않는다.
  - 확정

## 11. 삼세력 비밀 사건과 최종 각성 구조
- 사건
숨은 진실
  - 섬·어인 이해관계
  - 황색 교단 이해관계
  - 델타그린·카터 이해관계
  - 주안·엘리스 결과
  - 공개 시점
  - 확정/주의
- 대주교 구금
대주교는 모종의 거래와 반쯤 휴가를 겸해 자발적으로 구금됐다. 교단의 식인·괴이한 의식 문화와 성향이 맞지 않아 잠시 벗어나려는 개인적 이유도 있다.
  - 대주교 부재로 황색 교단의 현장 지휘가 분산되며, 섬은 일시적으로 방어할 시간을 번다.
  - 표면 목표는 대주교 구출이다. 현장 교단원들은 거래 전모를 모르고 생포·교환 명령만 수행한다.
  - 델타그린은 최고 전력을 구금해 협상 우위를 얻지만 거래의 세부는 공개되지 않는다.
  - 주연들은 초반에는 대주교 존재조차 모르고, 후반에 인질 사건의 진짜 목적만 이해한다.
  - 중반 단서 / 결말 직전 일부 공개
  - 거래 조건과 진짜 대가는 끝까지 설명하지 않는 맥거핀으로 유지한다.
- 엘리스·세실리아 인질화
황색 교단이 두 사람을 노리는 이유는 카터 가문에 대한 복수나 혈통 확보뿐 아니라, 대주교와의 인질 교환을 성사시키기 위해서다.
  - 카터 혈족이 섬에 들어오면 오래된 원한과 의식 조건이 동시에 활성화된다.
  - 엘리스 또는 세실리아를 살아 있는 교환 자산으로 확보해야 한다.
  - 가족을 구해야 하지만 대주교를 풀어주면 더 큰 초월 위협이 복귀한다.
  - 엘리스는 자신이 가족이 아니라 정치적·초월적 교환물이었다는 사실을 알게 된다.
  - EP2~10 부분 공개 / 후반 목적 공개
  - 생포 명령과 단순 납치를 구분해 복선 배치
- 카터의 상황 유도
윌리엄 카터는 섬의 비밀과 성맥, 엘리스의 정신계 능력 개화, 주안의 실험체 과거를 알고 의도적으로 두 사람을 사건 중심에 배치했다.
  - 카터 혈족과 실험체가 들어오며 의식·복수·권능 계승이 한꺼번에 움직이기 시작한다.
  - 카터가 만든 판을 이용해 대주교 구출과 성맥 확보를 동시에 노릴 수 있다.
  - 딸과 주안의 각성을 유도해 통제 가능한 초월 전력을 만들려 한다.
  - 두 사람의 만남과 위기, 일부 각성까지가 카터의 예상 범위였음이 드러난다.
  - 중후반 의심 / 결말 공개
  - 카터가 모든 변수까지 통제한 전지적 인물처럼 보이지 않게 실패와 오산을 남긴다.
- 블랙킹 계획
카터의 이상적 결과는 주안이 성맥의 힘으로 육체를 완성하고, 어인의 창에서 마법능력까지 얻어 ‘블랙킹’으로 각성하는 것이다.
  - 성맥과 어인의 창은 섬이 지켜온 핵심 자산이며, 한 인간에게 집중되면 종족의 생존 기반이 무너진다.
  - 완성된 블랙킹은 황색 교단에도 위협이자 포섭 가치가 있는 전력이다.
  - 블랙킹을 엘리스의 정신계 능력과 복종인자로 묶어 델타그린의 최강 전력으로 만든다.
  - 주안은 육체·마법·복종이 결합된 완성 병기가 될 예정이었다.
  - EP11~13 계획 공개
  - 블랙킹은 카터의 계획명이며 실제 완성 형태가 아님
- 불완전한 분할 각성
EP12에서 쿠바라가 패배하고 엘리스가 어인의 창을 노획한다. EP13 쇼거스의 마지막 핵이 성맥과 합쳐 카터의 블랙킹 계획처럼 모든 힘을 주안에게 집중시키려 할 때, 엘리스가 창을 붙들고 이안이 흐름을 분리한다. 주안은 성맥의 육체만 얻고 창은 엘리스의 정신계 능력과 강해지고 싶다는 집념에 공명해 옛 신의 권능을 넘긴다.
  - 섬의 두 핵심 힘이 한 사람에게 집중되지 않아 완전한 탈취는 실패한다.
  - 황색은 주안과 엘리스가 각각 불완전하지만 독립적인 초월 전력이 된 사실을 주목한다.
  - 윌리엄은 블랙킹 실패가 아니라 또 다른 성공으로 재평가하고 레드퀸 육성으로 방향을 바꾼다.
  - 주안은 성맥의 육체를, 엘리스는 정신계 능력과 어인의 창 마법을 얻는다.
  - EP12 창 노획 / EP13 쇼거스 마지막 핵
  - 쿠바라는 창 소유권 변화의 원인이고, 쇼거스 최종전은 분할 각성의 촉발점이다.
- 대주교 석방
쇼거스 최종전 뒤 황색 생존자들이 엘리스·세실리아와 거점 정보를 협상 카드로 삼고, 가론과 윌리엄의 대치·교환이 성립한 뒤 대주교가 구금에서 풀려난다.
  - 섬 입장에서는 황색 최고 전력의 귀환으로 새로운 위협이 시작된다.
  - 교단은 지도자를 되찾지만, 대주교 본인은 길었던 휴가가 끝난 듯 담담하게 복귀한다.
  - 윌리엄은 현장 손익과 기존 거래를 계산해 석방을 허용한다.
  - 엘리스는 협상과 부녀 대면을 거쳐 일시적으로 가족·주안과 재회한다.
  - EP13 가론-윌리엄 협상 뒤
  - 석방 전모는 맥거핀으로 남기되 협상 장면 자체는 생략하지 않는다.
- 이끌림의 진실 공개
카터 부녀의 대면과 세실리아·주안의 가족 재회가 끝난 뒤, 대주교는 주안에게 카터의 블랙킹 계획, 복종인자, 엘리스에게 느끼는 이끌림의 인위적 요소를 알려준다.
  - 섬의 힘이 인간 기관의 실험에 이용됐다는 사실이 드러난다.
  - 대주교는 카터 견제, 호기심, 심심풀이와 피해자인 주안을 돕고 싶은 마음으로 사실을 알려준다.
  - 카터의 설계가 폭로되며 딸과 주안에 대한 통제력이 흔들린다.
  - 주안은 자신의 감정과 선택이 어디까지 진짜인지 판단할 수 없게 된다.
  - EP13 가족 재회 직후
  - 원본 결말의 행복을 먼저 성립시킨 뒤 두 번째 감정 반전으로 사용한다.
- 주안의 이탈
원래 엘리스와 함께 가려던 주안은 자신의 과거, 성배 후유증, 육체 변화와 정신적 고통을 해결하기 위해 대주교를 따라간다.
  - 성맥의 육체를 가진 주안이 황색 교단 내부로 이동하면서 새로운 세력 균형이 생긴다.
  - 대주교는 주안을 치료·관찰하며 카터를 견제하고, 주안이 교단 안에서 반쯤 대놓은 감시자 역할을 하는 것을 허용한다.
  - 카터는 핵심 실험체를 잃지만 주안이 황색에 완전히 귀속되지는 않았다고 판단한다.
  - 엘리스는 이유를 듣지 못해 버려졌다고 느끼고, 주안은 감정의 진위와 인간성을 확인하려 거리를 둔다.
  - 에필로그
  - 주안은 황색 교단원이 아니라 내부 감시자다. 델타그린의 대의에는 공감하지만 양측의 과격한 방법을 모두 견제한다.
- 레드퀸 각성
윌리엄은 분할 각성을 또 다른 성공으로 평가하고, 주안과의 관계 파탄을 우려하면서도 엘리스를 레드퀸으로 육성한다. 엘리스는 델타그린의 육체 훈련과 라르고의 이능 훈련을 받으며 계속 주안을 찾는다.
  - 어인의 창의 힘이 인간 기관의 전력으로 전환되지만, 엘리스가 카터의 의도대로 움직일지는 별개다.
  - 황색은 강력한 정신·마법 전력의 탄생과 그녀가 주안을 추적하는 움직임을 경계한다.
  - 카터는 블랙킹의 대체품이 아니라 독립된 성공 사례로 레드퀸을 육성한다.
  - 엘리스는 상실을 수동적으로 견디지 않고 강해져서 주안을 직접 찾아가려 한다.
  - 후일담 / 차기부
  - 실연보다 자기결정·추적·능력 통제가 성장 동력이다. 육체 훈련은 델타그린, 이능 훈련은 라르고가 담당한다.
- 라르고 휴빙
윌리엄에게 충성하는 델타그린 최상위 이능력자이자 황색 대주교와 맞설 수 있는 히든카드다. 본편의 불개입은 능력 부족이 아니라 윌리엄의 명시적 관찰 명령이다.
  - 섬 세력은 그의 존재를 거의 모르며, 개입하면 전쟁 균형이 급격히 무너진다.
  - 대주교를 직접 상대할 수 있어 황색의 협상과 행동을 제한한다.
  - 다른 맥거핀성 임무를 수행하면서 섬 상황을 관찰한다. 엘리스 생존, 주안의 회복 가능성, 성맥·어인의 창이 황색에 완전히 넘어가지 않는다는 세 조건이 유지되는 동안 개입하지 않는다.
  - 감정은 희박하고 사무적이지만 엘리스에게 진심인 호감이 있어 그녀의 생존과 성장에는 관심을 둔다.
  - 후반 암시 / 차기부 본격 등장
  - 세 조건 중 하나라도 무너지면 윌리엄이 투입을 검토한다. 주연들의 승리가 라르고의 무능이나 방관 때문으로 보이지 않게 한다.

황색 교단의 최고 지도자. 소시민적인 성향과 달리 계약한 존재가 신화적 존재 ‘황색의 하스터’이므로 대주교직·교단·계약에서 완전히 벗어날 수 없다. 식인과 괴이한 의식을 싫어하지만 자리를 비우면 더 과격한 파벌이 교단을 장악할 수 있어 억제자이자 현실 정치가로 남는다.

  - 구금·인질 지령의 파편, 지나치게 편안한 수감 태도, 계약을 암시하는 황색 문양과 금지 조항
  - 결말 직전 일부 공개

  - 델타그린 / 카터

  - 실제로는 완성되지 않는다.
  - 확정

엘리스가 델타그린 정규훈련 이후 도달하는 각성명. 두 인격의 정신계 능력과 어인의 창 마법을 통합한 독립 전력이며, 떠난 주안을 다시 찾기 위한 힘이기도 하다.
  - 정신 간섭·통제·어인 마법·전장 지휘·두 인격의 협업
  - 엘리스 / 델타그린 / 라르고
  - 붉은 여왕·체스 문양·명령성 대사, 육체 훈련과 이능 훈련의 병행
  - 후일담
  - 윌리엄은 또 다른 성공으로 평가하지만, 엘리스는 아버지의 병기가 아니라 주안을 찾고 자기 힘을 통제하기 위해 훈련한다.

대전사장 쿠바라가 사용하던 실물 창. 정체는 노덴스 계열의 유물 ‘노덴스의 창’이지만, EP12에서 쿠바라가 패배하고 엘리스가 노획한 뒤에도 본문과 인물 인식에서는 어인의 창으로만 표기한다. EP13 쇼거스 최종전의 마지막 핵과 성맥이 결합하려는 순간 이안이 고서로 진명을 읽는다.
  - 어인계 주문·마력·정신계 능력과의 결합, 노덴스 계열 권능, 전대 사용자의 권능 계승
  - 섬·어인 / 쿠바라 / 엘리스 / 이안 / 노덴스 계열
  - EP12 쿠바라전·창 노획 / EP13 쇼거스 핵전·진명 공개·분할 각성
  - EP13 쇼거스 최종전

  - 동일 유물·전투 위계·공개 시점 수정 확정
- 라르고 휴빙
윌리엄에게 충성하는 델타그린 최상위 이능력자. 감정은 희박하고 사무적이지만 델타그린의 대의와 엘리스에 대한 진심인 호감을 지닌다.
  - 황색 대주교와 대등한 전투·봉쇄·상황 종결 전력, 엘리스의 이능 훈련 교관
  - 델타그린 / 카터
  - 작전 문서의 삭제된 인명, 윌리엄의 비상 연락선, 엘리스 상태 보고
  - 후반 또는 차기부
  - 본편에서는 윌리엄의 지시로 불개입하며, 후일담에서는 엘리스의 정신계·어인계 능력을 실전 수준으로 훈련한다.
  - 충성·불개입·교관 역할 확정
- 엘리스의 이중인격·정신계
외부 존재나 빙의가 아니라 원래 엘리스 안에 존재하던 두 인격의 발현. 둘 모두 자신을 같은 ‘나’로 인식하고 기본적으로 기억을 공유한다.
  - 표면 인격은 품위·관계·협상에 강하고, 내부 인격은 가문의 정신계 능력을 더 능숙하고 직접적으로 다룬다. 내부 인격은 표면 인격을 답답해하지만 자신을 보호하듯 엘리스를 보호한다.
  - 엘리스 / 카터 가문
  - 말투·자세·시선의 변화, 주저 없는 정신 간섭, 위기 시 ‘이번에는 내가 한다’는 식의 주도권 전환
  - 중후반
  - 서로를 별개의 타인이나 적으로 다루지 않는다. 기억 단절은 극심한 정신 공격 때만 예외적으로 발생하며, 평상시에는 같은 사건을 서로 다른 감정으로 해석한다.
  - 기억 공유·주도권 규칙 확정
- 황색의 하스터
황색 대주교와 계약한 신화적 존재. 대주교에게 압도적인 권능을 주는 대신 교단·직위·계약에서 완전히 벗어나지 못하게 하는 근원이다.
  - 계약 기반 권능, 금지 조항, 황색 교단의 신앙적 구속
  - 황색 교단 / 대주교
  - 황색 문양, 직접 거부하지 못하고 우회하는 대주교의 행동, 특정 명칭을 피하는 말버릇
  - 후속부까지 전모 비공개
  - 하스터의 의도와 계약 세부를 과도하게 설명하지 않는다. 대주교가 자유롭지 않은 이유만 독자가 이해하면 충분하다.
  - 계약 존재 확정 / 세부 맥거핀

  - 평소에는 기계적이고 사무적이다. 엘리스에게는 같은 무표정과 어조로 작업 멘트를 자주 던져 진담인지 농담인지 구별하기 어렵지만 실제로는 진심이다. 엘리스가 주안을 좋아한다는 사실을 알기 때문에 경쟁하거나 관계를 방해하지 않고, 교관과 연락책으로서 그녀의 성장과 행복을 바란다.
  - 윌리엄에 대한 충성과 엘리스의 자율성·행복이 충돌할 때, 명령을 정면으로 어기지 않으면서도 허용 범위의 정보와 선택지를 최대한 제공한다.
  - 직접 나타나 모든 문제를 해결하는 비상 카드가 아니라, 엘리스의 훈련·정보전·감정적 호흡을 담당하는 원격 조연

## 1부 연재 편성표 / 주요 EP·서브 EP 개요

운영 원칙
- 1부 본문은 제1화~제70화로 편성한다.
- 상위 탭에는 주요 EP와 서브 EP의 사건 흐름만 기록한다.
- 실제 원고는 하위 탭에서 5화 단위로 관리한다.
- 화마다 시작·종료 표기와 시점 화자 표기를 사용한다.

EP01 출항과 교수 — 제1화~제5화
- 주요 흐름: 네오 타이타닉호 출항, 금기의 고서 발견, 주안과 엘리스의 첫 관계, 황색 교단과 아킴의 접근.
- 서브 EP: 출항식과 교수 일행 / 읽히는 제목과 펜던트 / 첫 경호 업무 / 도박장의 노란 소매 / 아킴의 사격과 선상 지령.

EP02 식당의 소란과 침몰 — 제6화~제11화
- 주요 흐름: 선내의 일상 균열이 습격으로 확대되고, 각 인물이 서로 다른 목적을 품은 채 침몰 재난에 휘말린다.
- 서브 EP: 식당의 소란 / 수상한 승무원과 통신 두절 / 황색 교단 침투 / 선상 교전 / 구조 실패 / 침몰과 이산.

EP03 해안가의 재회와 날아다니는 페이지 — 제12화~제17화
- 주요 흐름: 조난자들이 섬의 해안에서 재회하고, 고서의 페이지와 섬의 이상 현상이 첫 단서를 제공한다.
- 서브 EP: 해안 표착 / 생존자 확인 / 주안과 엘리스의 재회 / 날아다니는 페이지 / 추적과 회수 / 섬 내부로의 이동.

EP04 동굴의 성씨와 카터 가문 — 제18화~제23화
- 주요 흐름: 동굴 탐사를 통해 섬의 혈통 기록과 카터 가문의 과거가 연결되며, 엘리스의 존재 의미가 달라진다.
- 서브 EP: 동굴 입구 / 오래된 이름 / 성씨가 남은 벽 / 카터 가문의 누락 기록 / 윌리엄의 짧은 회상 / 혈통과 의식의 연결.

EP05 제물 행렬과 절벽의 선택 — 제24화~제28화
- 주요 흐름: 제물 행렬을 목격한 일행이 구출과 추적 사이에서 갈라지고, 절벽에서 돌이킬 수 없는 선택을 한다.
- 서브 EP: 제물의 행렬 / 구출 논쟁 / 추적전 / 절벽의 선택 / 관계의 균열과 대가.

EP06 섬의 왕과 노란양복 — 제29화~제33화
- 주요 흐름: 섬의 통치자와 황색 교단의 목적이 충돌하고, 아킴은 조직의 명령과 자신의 판단 사이에서 흔들린다.
- 서브 EP: 섬의 왕 / 통치 구조 / 노란 양복의 집회 / 아킴의 이중 명령 / 불완전한 동맹.

EP07 위험구역과 페스타의 문 — 제34화~제38화
- 주요 흐름: 위험구역 진입과 페스타의 문을 통해 섬의 희생 규칙과 탈출 조건이 드러난다.
- 서브 EP: 금지 구역 / 변형된 생존자 / 페스타의 문 / 통과의 대가 / 문 너머의 단서.

EP08 드림랜드와 주안의 전투속행 — 제39화~제43화
- 주요 흐름: 드림랜드에서 기억과 현실이 뒤섞이고, 주안의 과거 실험과 비정상적인 전투 지속 능력이 암시된다.
- 서브 EP: 꿈의 진입 / 각자의 왜곡된 기억 / 어린 주안의 잔상 / 엘리스의 선택 / 전투속행 / 현실 귀환.

EP09 지도 탈취와 마도사의 소환 — 제44화~제48화
- 주요 흐름: 섬의 지도를 둘러싼 추적전과 마도사의 소환으로 주요 세력의 목적 차이가 표면화된다.
- 서브 EP: 지도 발견 / 탈취 / 추적과 역추적 / 마도사 소환 / 세력 간 충돌.

EP10 데이비드의 손과 페스타의 마지막 사명 — 제49화~제53화
- 주요 흐름: 데이비드와 페스타가 각자의 사명을 선택하고, 일행은 희생의 대가를 치르며 중앙신전으로 향한다.
- 서브 EP: 데이비드의 결단 / 잘린 손의 의미 / 페스타의 부탁 / 마지막 사명 / 중앙신전 진입.

EP11 중앙신전의 열과 빈 잔 — 제54화~제58화
- 주요 흐름: 중앙신전의 구조와 의식 도구를 조사하며, 비어 있는 잔이 누구를 기다리는지 추측하게 된다.
- 서브 EP: 신전의 열 / 각 열의 기록 / 비어 있는 자리 / 빈 잔 / 의식의 본체.

EP12 어인의 창과 마지막 구절 — 제59화~제63화
- 주요 흐름: 어인의 창과 고서의 마지막 구절이 봉인을 흔들고, 최종 선택의 조건을 완성한다.
- 서브 EP: 어인의 습격 / 창의 주인 / 마지막 페이지 / 마지막 구절 / 봉인 해제 직전.

EP13 폭풍의 눈 — 제64화~제70화
- 주요 흐름: 폭풍의 중심에서 최종 의식과 전투가 벌어지고, 인물들은 각자의 선택과 대가를 감당한 뒤 외전으로 이어진다.
- 서브 EP: 폭풍 진입 / 세력의 최종 계획 / 엘리스와 주안의 선택 / 이안의 해독 / 최종전 / 귀환과 후일담.
