## 2026-07-22 — Base 소설용 분화

- 사용: `PLAN → BUILD → REVIEW`
- 상황: 게임 개발용 Base를 소설 기획·집필·퇴고 저장소에 적용하되 기능 손실 없이 간소화할 필요가 있었다.
- 실제 변경: 공용 운영·정본·코어·가지치기·리팩토링·적대적 검토·동기화 기능을 5개 소설 Skill로 통합했다.
- 보존: 최소 Skill 자동 선택, 사용자 승인, 단일 책임 원본, 기능 보존, reference freshness, 증거 상태, 체크포인트.
- 제외: 게임 엔진·런타임·UI·성능 전용 기능.
- 추가: Canon·연표·지식 상태, 장면 카드, 다층 퇴고, 참고문장 저작권 경계, 독자 피드백 분류.
